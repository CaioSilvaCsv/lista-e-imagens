Compontentes de um PC

O que buscar?

Para uma busca rapida, se deve olhar:

  • Processador;
  • Memoria RAM;
  • Memoria rígida (HD);
  • GPU(Placa de video);

Qual escolher?


Para escolher, primeiro você tem que saber qual o teu uso, estudar, trabalhar, jogar,etc...
Após isso vai vendo o que cada um se adapta a ti!

Processador


Tente sempre comprar o mais recente, mas qual é melhor pra ti?
Processador tem varios, boa sorte, tu encontra um que lhe agrade.

Memoria RAM


Escolha 8 gb, só vai e confia.

Memoria rígida


É o tanto que vai guardar, guarda muita coisa, compre um bem grande.
Se possivel um SSD também ajuda na agilidade das tarefas.

Graphics Processing Unit (Placa de video)


Vai jogar ou trabalhar pesado? Eu indico, caso contrário, não há tanta necessidade.
